## Opsætning af Database
- Indsæt connectionstring parametre i "appsettings.json"
- Kør scriptet "All in one setup" uden for Visual Studio

*Note: i skal IKKE oprette en database på forhånd, dette bliver gjort automatisk.*
Når i kører All_In_One_Setup.SQL oprettes følgende automatisk:
- Database: Undy
- Relevante tables, views & procs
- Dummy data.
